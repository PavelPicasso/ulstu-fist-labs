Аудиторная нагрузка кафедр.

   Лекций - сумма лекционных часов. Суммируются лекционные часы только из тех планов, по которым в данный момент проводятся занятия. При подсчете считается, что лекции читаются для каждого потока отдельно.

   Лабораторных занятий - сумма часов лабораторных занятий . Суммируются часы лабораторных занятий только из тех планов, по которым в данный момент проводятся занятия. При подсчете считается, что для проведения лабораторных работ каждая группа делится на две подгруппы и для каждой подгруппы занятия проводятся отдельно.

   Практических занятий - сумма часов практичесих занятий. Суммируются часы только из тех планов, по которым в данный момент проводятся занятия. При подсчете считается, что практические занятия проводятся для каждой группы отдельно.
