Помощь. Аудиторная нагрузка кафедр.

   Количество читаемых дисциплин - это количество закрепленных за кафедрой дисциплин, по которым в данный момент проводятся занятия.

   Общий объем аудиторных часов - сумма аудиторных часов (лекций, лабораторных работ, практичесих занятий) по каждой из кафедр. Суммируются аудиторные часы только из тех планов, по которым в данный момент проводятся занятия. При подсчете учитывается количество групп и подгрупп. Считается, что лекции читаются для каждого потока отдельно. Практические занятий проводятся для каждой группы отдельно, а лабораторные для каждой подгруппы отдельно. Каждая группа разбивается на две подгруппы.

   Средний объем часов на одну дисциплину равняется Общему объему аудиторных часов разделенному на Количество читаемых дисциплин.
