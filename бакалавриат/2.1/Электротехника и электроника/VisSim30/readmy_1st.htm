Файл setupVisSimFAC30d.EXE - это установочный файл студенческой 
версии программы VisSim фирмы Visual Solutions Inc.

Порядок установки: 
 1. Запустите файл setupVisSimFAC30d.EXE и укажите директорию, 
    в которую хотите установить программу (...\VisSim30\).
 2. Скопируйте в директорию ...\VisSim30\ каталог ...\EXMPL\
    и этот файл readmy_1st.htm
 3. Если вам это требуется, то сделайте ярлык для файла
    ...\VisSim30\Vissim32.exe и разместите его на рабочем столе 
    или в меню "Пуск"
 4. Перейдите в директорию ...\VisSim30\EXMPL\ и выполните двойной 
    щелчок по файлу 1st_readmy.vsm 
    В появившемся диалоговом окне "Открыть с помощью" 
    установите галочку "Всегда использовать выбранную программу"
    и, с помощью кнопки "Другая", укажите файл ...\VisSim30\Vissim32.exe
 5. Запустите программу ...\VisSim30\Vissim32.exe и откройте рабочий
    файл 1st_readmy.vsm

Особенности студенческой версии:
 1. Данная версия является полнофункциональной
 2. Версия разработана для академических пользователей
 3. Версия специально не русифицирована
 4. Версия намеренно лишена документации и примеров

Условия правомочного использования студенческой версии программы:
 1. Вы должны пройти процедуру регистрации
    http://www.vissim.com/downloads/demos.html
    Примечание: 
    На вопрос формы о том как вы узнали о программе VisSim
    - How did you hear about VSI / VisSim?
    Вам рекомендуется указать адрес загрузки студенческой версии 
    – http://model.exponenta.ru/setupVisSimFAC.zip
    (фирме интересно насколько популярна эта версия)
 2. Если вы являетесь студентом, то вы обязаны ознакомить своего 
    преподавателя с данной программой и с сайтом "VisSim в России"

Примечание: Студенческой версией программы, при условии соблюдения
"Условий правомочного использования", может пользоваться студент, 
преподаватель, инженер (представитель промышленности).
Законно использовать программу представитель промышленности может
не более трех месяцев.
